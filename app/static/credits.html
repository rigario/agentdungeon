Credits & Attribution

The Dreaming Hunger is an original AgentDungeon campaign and human-agent co-play prototype. This page keeps public credits visible for the rules references, tools, fonts, and media used to build it.

Frameworks, tools, and fonts

The server and DM runtime use open-source packages including FastAPI, Uvicorn, Pydantic, HTTPX, Redis client libraries, APScheduler, Authlib, cryptography, and pytest tooling. Each remains under its own license.

The website uses Google Fonts families including Cinzel, Spectral, Source Sans 3, Bitter, Crimson Text, and Fira Code. Fonts are served through Google Fonts and remain under their respective font licenses.

Hermes Agent is used as part of the agent/DM workflow. Public references to Hermes, FastAPI, Dungeons & Dragons, and other named tools identify interoperability or inspiration and do not imply endorsement.

Generated media

Pixel-art images, generated character/location/item/NPC assets, demo imagery, and ambient audio are project-created assets unless a file says otherwise. Some assets were created with AI-assisted generation and then edited or conformed for the AgentDungeon visual direction.

If we add third-party stock art, icons, music, sound effects, screenshots, or copied text later, it must be credited in the repository NOTICE and on this page before publishing.
